# 🚫 No-Gos (harte Ausschlüsse)

Ein Kandidat, der ein No-Go verletzt, fliegt **unabhängig vom Match-Score** raus.
Jeder Eintrag braucht Herkunft (wann/wo gesagt) — nichts hier ist erschlossen.
