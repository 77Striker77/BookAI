# Originelle Prämisse

## Definition — wie ICH es meine

Ein **frischer, cleverer Grund-Einfall**, der das Werk trägt — ein „Was wäre wenn", das man so
noch nicht gelesen hat und das **in sich logisch durchdacht** ist. Bei [[Magic 2.0]]: „Die
Realität ist eine editierbare Computerdatei; wer das rausfindet, wird zum Gott/Zauberer." Der
Reiz ist die **Idee selbst** + ihre nachvollziehbare (hier: Nerd-/Programmier-)Logik. Abgegrenzt
von [[Ideenreichtum]] (dort: viele Einfälle über den Verlauf) — hier der EINE tragende Kern-Hook.
