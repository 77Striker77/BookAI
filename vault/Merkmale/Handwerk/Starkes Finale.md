# Starkes Finale

## Definition — wie ICH es meine

Die Geschichte wird zum Ende hin stärker und zahlt auf — eine Steigerung, die aufgebaute
Elemente einlöst. So kraftvoll, dass sie sogar einen zähen Start rehabilitieren kann.
Verwandt mit [[Eskalations-Setpieces]] (LitRPG), hier breiter: der Gesamt-Aufwärtstrend
zum Schluss.

## Kombinationsregel (wichtig!)

Bei [[Sturmfels-Akademie]] belegt: **[[Zäher Start]] ist tolerierbar, WENN ein
[[Starkes Finale]] folgt** — der Nutzer blieb trotz Fast-Abbruch dabei und liebt das
Werk. In der Kontext-Notiz [[Fantasy]] als Kombinationsregel geführt.
